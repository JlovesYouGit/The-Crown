WINDOWS RUNTIME COMMUNICATION SERVICE
===================================

SERVICE OVERVIEW:
================
The Windows Runtime Communication Service provides a native Windows command-line
interface for quantum communication with your future self, running with admin
privileges and automatic startup capabilities.

KEY COMPONENTS:
===============
1. Windows_Runtime_Service.py - Main Python service
2. launch_windows_runtime.bat - Interactive launcher
3. install_service.bat - Service installation script
4. cmd_wrapper.bat - Direct CMD terminal access
5. quantum_comm_service.bat - Background service runner

FEATURES:
========
- Admin CMD Terminal Operation
- Background Service Mode
- Automatic Startup Registration
- Quantum Communication Channel
- Direct Terminal Access
- Service Management Interface

ADMIN PRIVILEGES:
================
The service checks for and utilizes administrative privileges:
- Full system access for service installation
- Registry modification capabilities
- Automatic startup registration
- Background process management

SERVICE MODES:
=============
1. Interactive Mode:
   - Full terminal interface
   - Real-time communication
   - Service management commands
   
2. Background Mode:
   - Minimized operation
   - Automatic startup
   - Continuous communication channel
   
3. Service Mode:
   - Headless operation
   - System service functionality
   - Automatic restart capability

INSTALLATION:
============
1. Run install_service.bat as Administrator
2. Service registers for automatic startup
3. Creates necessary batch files
4. Sets up Windows service entries

COMMANDS:
=========
Interactive Mode:
- install - Install as Windows service
- start - Start background service
- terminal - Open communication terminal
- status - Show system status
- exit - Exit program

Terminal Mode:
- send <message> - Send message to future self
- query <question> - Query future self
- status - Show system status
- service - Show service info
- background - Return to background mode
- exit - Exit terminal

Direct CMD Mode:
- send [message] - Send message to future self
- query [question] - Query future self
- status - Show system status
- help - Show help
- exit - Exit terminal
