## Installation and Deployment

### Prerequisites
- Windows 10/11 or Windows Server 2016+
- .NET Framework 4.8
- Administrator privileges for installation
- Minimum 4GB RAM

### Installation Process
1. Execute `install_service.bat` as Administrator
2. All services automatically register with Windows Service Control Manager
3. Services start automatically on system boot
4. Verification through Windows Services panel or `verify_installation.bat`

### Configuration Options
- Separation distance (0.1-1.0mm)
- Yin-Yang balance ratio
- Reaction rate thresholds
- Hardware communication protocols
- BIOS optimization sensitivity
- Fan control algorithms
- Vacuum environment thresholds

## Troubleshooting

### Common Issues
1. **Service fails to start**
   - Ensure running as Administrator
   - Check .NET Framework 4.8 installation
   - Verify service dependencies
   - Check ASUS Armory Crate installation (for Armor BIOS service)

2. **Poor energy separation performance**
   - Check system load levels
   - Verify hardware communication
   - Review log files for errors

3. **BIOS optimization issues**
   - Verify ASUS motherboard compatibility
   - Check BIOS communication
   - Ensure proper BIOS settings

### Uninstallation
To remove the services:
1. Run `uninstall_service.bat` as Administrator
2. Or use individual uninstall scripts for selective removal
3. Or use PowerShell scripts for selective removal
